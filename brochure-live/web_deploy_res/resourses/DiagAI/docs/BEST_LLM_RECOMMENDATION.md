## 🎨 Why NOT Vision Models?

You might think you need vision (like LLaVA-NeXT-Video) for "display chart map", but you don't:

- **Your charts are generated by Python tools** (matplotlib)
- The LLM needs to **call the tool**, not see the chart
- Vision models are **heavier** and **slower** for no benefit
- **Tool Use** is what matters, not vision
