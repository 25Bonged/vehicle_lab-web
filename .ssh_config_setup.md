# SSH Key Setup for GitHub

An SSH key has been generated for you. To complete the setup:

## Step 1: Add SSH Key to GitHub

1. Copy the SSH public key displayed above
2. Go to: https://github.com/settings/keys
3. Click "New SSH key"
4. Give it a title (e.g., "MacBook - Vehicle Lab")
5. Paste the public key
6. Click "Add SSH key"

## Step 2: Test SSH Connection

After adding the key, test it with:
```bash
ssh -T git@github.com
```

## Step 3: Push to GitHub

Once SSH is set up, you can push with:
```bash
git push -u origin main
```

---

**Note:** The SSH key file is located at: `~/.ssh/id_ed25519_github`
